Lesson one: Basics of the web and HTML

The Web

  • Invented 1990’s - now has 30 billion webpages, it's basically a collection of HTML documents
  • HTML = Hypertext markup language: HTML language is what glues everything together on the web - text, videos, images
  • The links between webpages is what makes the web “weblike”

Major Pieces of the web

The web is basically made up of three components: You, your computer and your browser; the internet, and servers.
  • The browser makes requests via the internet to servers with a protocol called HTTP
  • The server responds with files that your browser displays - browsers include IE, Chrome, Firefox
  • Servers are computers just like yours, but are optimized for hosting files instead of browsing

HTML

A. HTML docs are the heart of the web and are made up of:
  • Text content - what you see
  • Mark up - what it looks like or how it is arranged (not seen)
  • References to other docs (images, videos)
  • Links to other pages
B. HTML markup

HTML markup is made up of tags, which are constructed like this: <opening tag>content</closing tag>

Examples:
  • <b> tag - makes content bold
  • <em> tag - means emphasis - makes content italicized
  • <a> tag - anchor tag - this is for making links
C. HTML attribute tags

An HTML attribute is a modifier of an HTML element. They are usually added to a HTML start tag, which along with content and end tag comprises most HTML elements. Particular attributes are only supported by particular element types; if they are added to elements that do not support them they will not function (from the HTML attribute Wiki page).

For example:

<tag ATTR = “value”>contents</tag>

Examples:
  • <a href = “www.reddit.com”>derp </a> = derp
  • <a href = “www.w3c.com”> HTML </a> = HTML
  • <a href = “www.udacity.com”>my favorite </a> = my favorite
D. Image tag

<img> is used to download images. For example:
  • <img src = “url” alt = “text”>
“url” is for the image to download
“text” is for broken requests - the text that is displayed if image does not load
  • <img src = “http://www.udacity.com/cs253x/hipmunk.png” alt> = “hipmunk!”
The image tag does not have or need a closing tag - it’s a “void tag” - has no content

E. Whitespace

When you enter text in multiple lines, in HTML it renders it in single space <br> is to create a space break. This is another example of a void tag.
  • To make double spaces you can use either a <br> or a <p> tag.
  • To use <br> as a double space, you just use two in a row: <br> <br>
<p> is a “block text” - you need a closing tag: <p> narrative </p>

Difference between "br" and "p":
  • "br" is an “inline element” - just ends a line
  • "p" narrative is a “block element” - it makes an invisible box to put narrative. The box has width and height.
F. Span/Div Elements

The "Span" and "Div" tags just contain their content:
  • Span is inline and is written like this: <div> text </div>
  • Div is block and is written like this: <span> text </span>
You can add attributes called “class” to Span/Div elements:
  • <div class = “foo”> text </div>
  • <span class = “bar”> text </span>
